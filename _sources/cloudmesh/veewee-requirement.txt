Veewee requirements
====================

`requirement.md <https://github.com/jedi4ever/veewee/blob/master/doc/requirements.md>`_

On Ubuntu 14.04
---------------

::

  apt-get install ruby ruby1.9.1-dev libxslt1-dev libxml2-dev zlib1g-dev

On Mac OS X
-----------

::

  use homebrew to install apple-gcc42, build-essential

Cloudmesh Cluster
======================================================================

Overview
----------------------------------------------------------------------

This lesson will introduce you to the cloudmesh cluster command.

It supports the creation of a virtual compute cluster.  The command
``cluster`` is used to start, configure, manage or update a number of
virtual machines. Details about the command can be found via the cm
help command and in the section :ref:`cloudmesh-cm-cluster-ref`

.. tip:: Duration: 1 hour

Prerequisite
----------------------------------------------------------------------

* You must have cloudmesh installed.
* You must have access to a cloud that is configured within cloudmesh
  
Exercises
----------------------------------------------------------------------

Exercise CM.CLUSTER.1
^^^^^^^^^^^^^^^^^^^^

Install a virtual cluster with 3 vms

Exercise CM.CLUSTER.2
^^^^^^^^^^^^^^^^^^^^

Install a virtual cluster on which you  run hadoop.

Exercise CM.CLUSTER.3
^^^^^^^^^^^^^^^^^^^^

Install a virtual cluster on which you  run a distributed version of mongodb.

Report
======================================================================

This lesson describes the requirements for submitting reports.

Reports will be accepted only under the following conditions:

* IEEE transaction format (IEEEtrans)
* RST format (RST)

See below for details.

.. important::
   Documents submitted that do not conform to IEEEtrans or RST will
   **NOT** be graded.


IEEE transactions format
----------------------------------------------------------------------

Templates for the IEEE transactions format can be found online
at the `IEEE website`_.
If a document is prepared using IEEEtrans you must submit all relevant
files.

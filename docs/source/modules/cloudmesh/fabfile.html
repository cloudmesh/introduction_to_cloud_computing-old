fabfile.test module¶

fabfile.test.info[source]¶

list all functions of file with the partial name f :param f: the name will be test_<f>.py

fabfile.test.start[source]¶

executes a test with a given partial filename and partial name of the test class. the first function in the test file will be returned. for example :

fab t:inventoy,clean

will look in the file test_inventory.py for the first function that has ‘clean’ in its name and execute that test. The program does not support

looking only for @task members.
Parameters:
  • f – the partial filename
  • name – the partial name of the test task

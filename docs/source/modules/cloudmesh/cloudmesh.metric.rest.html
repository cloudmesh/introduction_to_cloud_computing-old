class cloudmesh.metric.rest.cm_metric_rest.CMMetricREST[source]¶

Command Line Interface for fg-metric.

Usage: fg-metric-cli

As a cli version of fg-metric, this module provides usage data with search options.

  • Excutable name is fg-metric-cli (defined by setup.py).
  • CM Cloud Mesh would be one of the examples using fg-metric-cli.
{

“start_date” : start date of search (datetime), “end_date” : end date of search (datetime), “ownerid” : portal user id (str), “metric” : selected metric name (str), “period” : monthly, weekly, daily (str), “clouds” : set of clouds (list)

[
{ “service” : cloud service name (str),
“hostname” : hostname (str), “stats” : value (int) } ... ]

}

Example 1. Get user statistics

$ fg-metric-cli -u hrlee

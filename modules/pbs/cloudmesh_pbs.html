class cloudmesh_pbs.cmd3_check.cmd3_check[source]¶
static yaml(plugin)[source]¶

This function reads in the cloudmesh cmd3.yaml file and tests if the requested plugin module is included in the yaml file.

an example plugin name os “cloudmesh_pbs” to which automatically a .plugins will be appended if not specified.

Returns:True if the plugin is in the yaml file

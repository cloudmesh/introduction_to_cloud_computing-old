cloudmesh.metric.openstack.migrating_mysql_to_mongodb module¶

cloudmesh.metric.openstack.migrating_mysql_to_mongodb¶

This module migrates mysql database to mongodb.

class cloudmesh.metric.openstack.migrating_mysql_to_mongodb.Migrate_MySQL_to_Mongo[source]¶

The Migrate_MySQL_to_Mongo exports mysql database tables in a csv format, and pour into mongodb using mongoimport command line tools. The database information is loaded from ‘dbinfo.yaml’ file which contains db access information such as hostname, user name, and password.
